== Description ==

The Plus One plugin easily adds the Google +1 button to your site.

Watch this video to learn more about the +1 button:
[youtube http://www.youtube.com/watch?v=OAyUNI3_V2c]

Features:

* Button Preview
* Manual Code Generator
* Include button on Posts?
* Include button on Pages?
* Button location (above, below or both)
* Button Size
* Language
* Advanced options
* Localization

== Installation ==

1. Upload the Plus One folder to the plugins directory of your WordPress installation
2. Activate the plugin
3. Navigate to the Plus One submenu under 'Settings' from your dashboard
4. Configure the plugin to your desired settings
